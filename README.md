<p align="center">  It's a simple front-end comment system, where a user can login using google login and then redirected to home page where user can see all the comments and reply or like a particular comments.
User can even create his or her comment and can modify, delete, reply or like comments.
Once you is done with the application then user can logout and move to login page again.

# Features:


    ‚ùñ Login : (Login With Google) 
        ‚Ä¢ Users needs to login using google account. 
        ‚Ä¢ Once user is logged in then user will be redirected to home page.

    ‚ùñ Home Page :   
        ‚Ä¢ Home Page has two section
            ‚Ä¢ Profile Page
                ‚Ä¢ Once user is logged in then user will be able to see his or her
                    ‚Ä¢ Profile Picture
                    ‚Ä¢ Name 
                    ‚Ä¢ Email
            ‚Ä¢ Create Comment
                ‚Ä¢ User can start a comment by clicking on "Create" button.
            ‚Ä¢ View Comments   
                ‚Ä¢ All the comments will be loaded by date wise sorted order.
            ‚Ä¢ CRUD Operation:
                ‚Ä¢ Read : User can read comment of other user's.
                ‚Ä¢ Like : User can like other user's comment or can like his or her comment as well.
                ‚Ä¢ Reply : User can reply to other user's comment and comment can be nested and there is no limit.
                ‚Ä¢ Update : User can only update his or her comments and will not be able to modify other user's comment.
                ‚Ä¢ Delete : User can only delete his or her comments and will not beable to delete other user's comment.
    ‚ùñ Footer Page :
        ‚Ä¢ Footer Section will have some details like copy right, privacy policy etc.

    ‚ùñ Logout Page :
        ‚Ä¢ Once user is done with the application and then user can logout and user will be redirected to login page, where user can login again.
